# React Timer & Alarm Application ⏰

A feature-rich React application that combines three essential time management tools in one elegant interface.

## ✨ Features

### 🕐 Digital Clock
- Real-time display with HH:MM:SS format
- Updates every second using Moment.js
- Clean, modern design

### ⏳ Countdown Timer
- Customizable hours, minutes, and seconds input
- Start, Pause, Resume, and Reset controls
- Input validation (hours ≤ 99, minutes/seconds ≤ 59)
- Automatic sound alert when timer reaches zero
- Visual countdown display with leading zeros

### 🔔 Alarm System
- Easy time picker for setting alarms (HH:MM)
- Set/Clear alarm functionality
- Real-time clock monitoring
- Plays sound when alarm time matches current time
- Browser notifications support
- Automatic alarm deactivation after ringing

## 🛠️ Technologies
- React (Hooks: useState, useEffect, useRef)
- Moment.js for time manipulation
- Howler.js for cross-browser audio
- CSS3 for responsive styling
